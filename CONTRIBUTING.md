# Contributing

Thanks for helping improve ClawTasker CEO Console.

## Development flow

1. Fork the repository.
2. Create a feature branch.
3. Run the server locally and test the UI.
4. Run `python -m unittest discover -s tests`.
5. Update `CHANGELOG.md` and `VERSION` when your change is release-worthy.

## Commit guidance

- Keep commits focused.
- Document API or schema changes.
- Preserve secure defaults.
- Do not add dependencies unless they materially improve the product.
