ClawTasker is an open-source, local-first CEO console for managing teams of human and AI agents — sprint board, virtual office, REST API, and zero Node.js required.

View on GitHub REST API Docs

Everything a CEO needs to direct an AI team

Virtual Office

Live 2D office floor with animated agent sprites. See who's working, blocked, in a meeting, or offline — in real time.

Sprint Board

Kanban board with Backlog → Ready → In Progress → Validation → Done. Story points, epics, task links, and acceptance criteria.

Mission Planner

Create mission briefs with success criteria, staffing, dependencies, and risk tracking. Assign missions to agents directly.

REST API for AI Agents

Every GUI action has an API equivalent. Agents register, send heartbeats, create tasks, post comments, and publish run results.

Dashboard

CEO overview with attention queue, mission brief, staffing radar, risk indicators, and sprint velocity at a glance.

Zero Dependencies

Python stdlib only. No Node.js, no npm, no Docker. Run with python3 server.py and open a browser.

Works with any AI agent

Claude / Claude Code OpenAI / GPT Gemini Any HTTP-capable agent Python agents Custom LLM wrappers OpenRouter Local models

Quick Start

# 1. Clone the repo
git clone https://github.com/erranthex/clawtasker.git
cd clawtasker

# 2. Set your API token
export CLAWTASKER_API_TOKEN="your-secret-token"

# 3. Start the server
python3 server.py

# 4. Open in browser
open http://localhost:3000

Or open ui/dist/index.html directly — no server needed for the demo mode.
